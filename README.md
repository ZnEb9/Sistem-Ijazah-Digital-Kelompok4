# 🎓 Sistem Penerbitan dan Verifikasi Ijazah Digital Terdesentralisasi
**Proyek Ujian Tengah Semester (UTS) Mata Kuliah Teknologi Blockchain**  
**Program Studi Sistem Informasi - Institut Teknologi PLN**

Sistem berbasis *smart contract* ini dirancang untuk mencegah pemalsuan dokumen akademik dengan memanfaatkan teknologi blockchain. Proyek ini mendemonstrasikan bagaimana institusi pendidikan dapat menerbitkan ijazah secara *immutable* (tidak dapat diubah) dan bagaimana pihak ketiga (seperti perusahaan) dapat memverifikasi keaslian ijazah tanpa harus bergantung penuh pada *database* terpusat universitas.

## 🚀 Status Proyek
Saat ini proyek berada dalam **Tahap UTS** (Fase Perancangan & Analisis)[cite: 1]. 
Fokus utama pada tahap ini adalah justifikasi penggunaan teknologi blockchain untuk studi kasus ijazah dan perancangan *skeleton smart contract* yang mencakup *access control* dan *event logging*[cite: 1]. Implementasi penuh dan interaksi *front-end* akan dilanjutkan pada tahap UAS.
